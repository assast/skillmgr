# Icons Directory

This directory should contain the following icon files for Tauri:

- `32x32.png` - 32x32 pixels PNG icon
- `128x128.png` - 128x128 pixels PNG icon
- `128x128@2x.png` - 256x256 pixels PNG icon for high DPI displays
- `icon.icns` - macOS icon format
- `icon.ico` - Windows icon format

You can generate these icons using the Tauri CLI:
```bash
npm run tauri icon /path/to/your/source/icon.png
```

For now, these are placeholder files to make the build process work.
